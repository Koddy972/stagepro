🐛 EN CAS DE PROBLÈME
-------------------------------------------

Problème : Serveur ne démarre pas
└─ Solution : Vérifier que le port 8000 est libre

Problème : Redirection vers /client/login
└─ Solution : Se connecter en tant que client

Problème : Erreur CSRF Token
└─ Solution : php artisan cache:clear

Problème : Erreur Stripe API
└─ Solution : Vérifier les clés dans .env
